## 📘 Lista de Exercícios de Programação I

Este repositório contém a resolução da **Lista de Exercícios de Programação I**, proposta pelo Centro Universitário do Distrito Federal (UDF), na disciplina de **Programação de Computadores**.

Os exercícios foram desenvolvidos com foco em consolidar os conceitos fundamentais da programação utilizando Python.

---

## 🧠 Conteúdos Abordados

A lista está dividida em três níveis de dificuldade:

### 🔹 Nível 1 — Fundamentos

* Entrada e saída de dados
* Operações matemáticas básicas
* Cálculos simples

### 🔹 Nível 2 — Aplicações e Fórmulas

* Cálculos mais complexos
* Manipulação de variáveis
* Porcentagens e conversões
* Uso de divisão inteira e resto

### 🔹 Nível 3 — Lógica e Condições

* Operadores relacionais
* Operadores lógicos (`and`, `or`)
* Validações e expressões booleanas

---

## ✨ Diferenciais

* Exercícios organizados por nível de dificuldade
* Código padronizado e comentado
* Foco em boas práticas para iniciantes

Cada pasta representa um nível da lista de exercícios:

* 🔹 **Nível 1** → Entrada, saída e aritmética básica (ex01–ex10)
* 🔹 **Nível 2** → Atribuições e fórmulas (ex11–ex20)
* 🔹 **Nível 3** → Operadores lógicos e relacionais (ex21–ex30)

Cada arquivo contém:

* 📌 Enunciado do exercício
* 💻 Implementação em Python

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/lista-programacao-python.git
```

2. Acesse o diretório:

```bash
cd lista-programacao-python
```

3. Execute um exercício:

```bash
python exercicios/exercicio01.py
```
